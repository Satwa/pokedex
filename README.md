PHP exercise given by my teacher at [HETIC](https://hetic.net) to learn to master PHP and queries (here with [PokeAPI](https://pokeapi.co)).

### Run the project

You require at least PHP 7.0.

Go to project directory and run once:

```composer install```

Then enjoy!
